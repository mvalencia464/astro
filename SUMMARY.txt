CORE CONCEPT (30 SECONDS)
================================================================================

REACT (Heavy):
  const [count, setCount] = useState(0);
  <button onClick={() => setCount(count + 1)}>{count}</button>
  
  Issues:
  - React runtime (85KB)
  - Virtual DOM diffing
  - Re-rendering overhead
  - Initialization time

ASTRO (Fast):
  <button id="btn">0</button>
  <script>
    let count = 0;
    document.getElementById('btn').addEventListener('click', () => {
      count++;
      document.getElementById('btn').textContent = count;
    });
  </script>
  
  Benefits:
  - No runtime (0KB)
  - Direct DOM updates
  - Only code you write
  - Instant page load

Result: 5.5x faster, 85% less JavaScript

    → Check element IDs match

  "Lighthouse score low"
    → Check for missing image dimensions
    → Look for layout shift
    → Check for console errors

  "Touch doesn't work on iOS"
    → Add "touch-action: manipulation" CSS
    → Use addEventListener (not inline handlers)
    → Test on real device
